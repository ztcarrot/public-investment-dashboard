# 使用 akshare 获取债券数据

## 概述

已成功集成 akshare 库，用于获取债券类型资产的历史数据。

## 实现细节

### 1. 依赖添加

在 `requirements.txt` 中添加：
```
akshare>=1.12.0
```

### 2. 数据获取方法

**文件**：`utils/data_fetcher.py`

**新增方法**：`get_bond_historical_from_akshare()`

- 使用 `ak.bond_china_yield()` 获取中债国债收益率曲线数据
- 根据10年期国债收益率推算债券价格
- 简化定价公式：`价格 = 100 * (1 - (收益率 - 0.02) * 3)`
- 价格限制在合理范围：90-110 元

**数据流程**：
```
债券类型资产
  ↓
优先尝试 akshare（国债收益率曲线）
  ↓
失败则回退到东方财富基金API
```

### 3. 特殊处理

**19789/019789（25特国06）**：
- 优先尝试 akshare
- 失败后使用原有的专用方法（基于当前价格推算历史数据）

**其他债券（如005350短债基金）**：
- 优先尝试 akshare
- 失败后使用东方财富基金API

## 优势

1. **数据来源多样化**：不依赖单一API，提高数据获取成功率
2. **收益率驱动**：基于市场收益率推算价格，更反映市场实际情况
3. **自动回退**：akshare 失败时自动使用备用方法
4. **代码健壮**：完善的错误处理和日志记录

## 注意事项

1. **数据精度**：akshare 提供的是国债收益率曲线，推算的价格是近似值
2. **适用范围**：最适合国债类资产，对于债券型基金建议使用东方财富API
3. **日期限制**：收益率曲线数据可能不包括所有交易日
4. **价格波动**：推算的价格波动可能小于实际市场波动
